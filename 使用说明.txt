🎨 AI智能制图工作室 - 使用说明

这是一个AI制图网站，可以根据你的文字描述生成图片。

📁 文件说明：
- index.html : 网站的主页面，包含所有的界面元素
- app.py : Python后端程序，处理用户请求和图片生成
- requirements.txt : 需要安装的Python库列表
- 使用说明.txt : 这个文件

🚀 如何启动网站：

第1步：安装Python库
在终端（命令行）中运行：
pip install -r requirements.txt

解释：这条命令会自动安装网站需要的所有Python库，主要是Flask框架。

第2步：启动服务器
在终端中运行：
python app.py

解释：这条命令会启动网站服务器，你会看到类似这样的提示：
"Running on http://127.0.0.1:5000"

第3步：打开网站
在浏览器中访问：http://localhost:5000

🖥️ 网站功能：

1. 文字描述输入框
   - 在这里输入你想要生成的图片的详细描述
   - 比如："一只可爱的小猫坐在花园里"

2. 风格选择
   - 真实摄影：像真实拍摄的照片
   - 迪士尼风格：卡通动画风格
   - 扁平风格：简约的平面设计
   - C4D渲染：立体的3D效果
   - 超写实CG：非常逼真的电脑图像

3. 上传参考图片（可选）
   - 你可以上传一张图片作为参考
   - 支持JPG、PNG等常见格式
   - 文件大小不能超过10MB

4. 生成按钮
   - 点击后开始生成图片
   - 需要等待一段时间（现在是模拟过程）

⚠️ 重要提醒：

1. 这个版本是演示版本，目前生成的是示例图片
2. 如果要接入真正的AI图片生成功能，需要：
   - 注册AI服务（如OpenAI、Stability AI等）
   - 获取API密钥
   - 修改app.py中的simulate_image_generation函数

3. 确保你的电脑已经安装了Python（版本3.7以上）
4. 如果遇到问题，检查终端中的错误信息

💡 技术架构：
- 前端：HTML + CSS + JavaScript（纯前端技术）
- 后端：Python + Flask框架
- 文件处理：支持图片上传和预览
- 响应式设计：支持电脑和手机访问

🛠️ 如果想要扩展功能：
- 可以添加用户注册登录
- 可以保存生成历史
- 可以添加更多风格选项
- 可以接入真正的AI图片生成API

祝你使用愉快！ 🎉
